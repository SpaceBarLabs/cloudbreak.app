Frequently Asked Questions

What is FreshRSS?

FreshRSS is a self-hosted RSS feed aggregator that lets you follow your favorite websites and blogs in one place. We host and maintain FreshRSS for you, so you don't have to worry about server management or updates.

Can I install extensions?

We currently do not support installing custom extensions for security and stability reasons. However, if you have a specific extension request, you can submit it through our feature request form. We regularly review these requests and may add popular extensions in the future.

What apps do you recommend for reading FreshRSS?

For Android users, we recommend Capy Reader. For Linux desktop users, NewsFlash is an excellent choice. Both apps work great with FreshRSS and offer a smooth reading experience. Any compatible app should work as expected. You can also refer to the official list of FreshRSS compatible apps.

How do I set up mobile app access?

To use FreshRSS with mobile apps, you'll need to set up an API password, which is separate from your main user password. After logging in, go to your settings and set an API password. You'll use this API password (not your regular login password) when connecting through mobile apps.

How long does account creation take?

In most cases, your account will be created quickly after your subscription is processed. However, in some cases, it may take up to two business days to complete the account setup.
