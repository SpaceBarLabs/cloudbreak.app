What is {{ product.name }}?

Feedbin Mini is a modern, fast RSS feed reader that lets you follow your favorite websites and blogs in one place. We host and maintain {{ product.name }} for you, so you don't have to worry about server management or updates.

What makes it "mini"?

{{ product.name }} is a simplified version of the full Feedbin service. It includes the core features of Feedbin, but excludes some advanced features like newsletters, full content extraction, preview image cropping, and the like. This makes it a more affordable option for users who only need the essentials, but still appreciate the design and function of Feedbin, as well as its general native app compatibility.

What apps do you recommend for reading {{ product.name }}?

For Apple platforms (Mac, iOS, iPad), we recommend NetNewsWire. For Android users, we recommend Capy Reader. For Windows users, we recommend Fluent Reader. For Linux desktop users, NewsFlash is an excellent choice. These apps work great with Feedbin and offer a smooth reading experience. Any compatible app should work as expected.

What kind of support do you provide?

We focus on providing a reliable, high-uptime {{ product.name }} hosting service. Our support is mostly limited to the information provided in this FAQ. While we maintain and update the Feedbin installation, we do not provide technical support or tutorials for using Feedbin itself - for that, please refer to the official Feedbin documentation.

You can submit feature requests through our feature request form. We regularly review these requests and may add popular features in the future.
